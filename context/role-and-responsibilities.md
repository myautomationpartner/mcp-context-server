## My Automation Partner — Sole Creator (Part-Time, ~20 hrs/week)

- Designing and building the network infrastructure the platform runs on (currently complete and operational)
- Architecting the application layer for the customer portal
- Building automation workflows (n8n) that power the platform's features
- Managing the full deployment pipeline — GitHub to Cloudflare Pages for web/portals
- Database design and management in Supabase
- API integrations (social media via Zernio, etc.)

- Managing AI context continuity — maintaining plan.md and creating handoff documents to move between AI agents/sessions
