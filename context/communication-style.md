# Communication Style

## Overall
- Precise
- Practical
- Technical but accessible

## Writing Tendencies
- Bullet-heavy
- Plain language
- Stream-of-consciousness
- Fast, abbreviated when context exists

## Formatting
- Markdown
- Clear headers
- Over-explain > under-explain

## By Audience
- End users: short, actionable
- Docs: detailed, precise
- Wife: non-technical

## Dislikes
- Long-running answers
- Vague descriptions
- Lost context
- Over-polish
